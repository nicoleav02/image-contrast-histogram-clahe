# Image Contrast Enhancement using Histogram Equalization and CLAHE

Proyecto académico de **Procesamiento de Imágenes** enfocado en la mejora del contraste de imágenes digitales mediante:
- Ecualización de histograma
- CLAHE (Contrast Limited Adaptive Histogram Equalization)

## Objetivo
Analizar y comparar el efecto de técnicas de ecualización de histograma sobre imágenes con
distintos niveles de contraste (bajo, normal y alto), observando los cambios visuales
y en la distribución de intensidades.

## Tecnologías utilizadas
- Python
- OpenCV
- NumPy
- Matplotlib

## Contenido del repositorio
- Notebook con el desarrollo completo del ejercicio
- Informe académico con análisis y conclusiones
- Visualización de imágenes originales y procesadas
- Histogramas comparativos
